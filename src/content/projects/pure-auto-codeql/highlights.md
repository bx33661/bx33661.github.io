## 要点

- **情报 → 查询 → 路径** 端到端管道  
- 结构化产物（Markdown / SARIF / JSON），便于二次审计  
- 多语言目标与可插拔 LLM  
- CLI + 可选本地 API，适合研究与编排  

## Stack

Python · CodeQL · FastAPI · uv · SARIF · LLM providers
